Homebrew formulae for [CardPointers](https://cardpointers.com) tools.

## Install

```bash
brew tap cardpointers/tap
brew install cardpointers
```

## Update

```bash
brew update
brew upgrade cardpointers
```

## What is CardPointers?

[CardPointers](https://cardpointers.com) helps you maximize your credit card rewards by telling you which card to use for every purchase. The CLI brings your wallet to the terminal — perfect for quick lookups, scripting, and AI agent integrations.

👉 **[CLI documentation & source](https://github.com/cardpointers/cli)**
